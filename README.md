# AI Visual Speech Recognition

### About
- An AI visual speech recognition that interprets what an individual is saying by only analyzing their facial movements, effectively transcribing their speech without relying on the audio.
- Achieves an accuracy of 86.4% by using a deep learning neural network and incorporating computer vision.
- The model is transformed into a full stack application, enabling users to select from a predetermined set of videos for real time transcription.
